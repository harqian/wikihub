Agent Protocols

Agent communication protocols define how autonomous software agents exchange structured messages, negotiate capabilities, and coordinate actions. The two dominant patterns emerging in 2026 are MCP (Model Context Protocol) and A2A (Agent-to-Agent).

MCP focuses on tool registration and invocation — a client discovers a server's capabilities via a manifest and calls tools with typed parameters. A2A is higher-level: agents exchange "tasks" with lifecycle semantics (pending, working, completed, failed).

Both protocols assume HTTP transport but differ on session semantics. MCP uses stateless request-response; A2A supports long-running tasks with streaming updates via SSE.
